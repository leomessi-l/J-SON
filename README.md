JSON (JavaScript Object Notation) es un formato ligero y estándar basado en texto para el intercambio de datos, independiente del lenguaje de programación. Utiliza una sintaxis sencilla de pares clave/valor y arreglos, siendo fácil de leer para humanos y eficiente para máquinas, ideal para APIs y configuración. 

Estructura: Se basa en colecciones de pares nombre/valor (objetos, entre {}) y listas ordenadas de valores (arreglos, entre []).

Tipos de datos: Soporta cadenas de texto (entre comillas dobles), números, booleanos (true/false), null, objetos y arreglos.

Compatibilidad: Aunque deriva de JavaScript, es compatible con casi todos los lenguajes modernos (Python, PHP, Java, C#, etc.).
Uso: Principalmente para enviar datos entre servidor y aplicaciones web/móviles
